# 项目中保持编码的风格一致

## 1. 需要安装插件**EditorConfig for VS Code** 支持

## 2. 生成 **editorconfig** 配置

![editorconfig配置](/frontend/engineering/example/editorconfig.jpg)


```
# .editorconfig

root = true

[*]
charset = utf-8
indent_style = space
indent_size = 2
end_of_line = lf
insert_final_newline = true
trim_trailing_whitespace = true
```
